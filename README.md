# Task Man

A full-stack todo application built with **Django**, **htmx**, and **Tailwind CSS**. *It's nothing more than a learning project*.


## Stack

- Django
- htmx
- Tailwind CSS
- daisyUI


## Prerequisites

- [uv](https://github.com/astral-sh/uv) (for dependency management)
- Python 3.12+


## Getting Started

1. Clone the repository: `git clone https://github.com/risvan-mhd/task-man.git`
2. Go to the project directory: `cd task-man`
3. Sync dependencies: `uv sync`
4. Run migrations: `uv run python manage.py migrate`
5. Start the server: `uv run python manage.py runserver`
